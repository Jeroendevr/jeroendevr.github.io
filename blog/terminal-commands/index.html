Terminal Commands

written by Jeroen on 2019-10-23

Usable for Bash and Zsh (macOS Catalina and higher) Terminal commands

  • cd Change directory
  • cd.. Change directory one level up
  • ls List directory

Can be combined with functions to add specific action For example

  • ls -a Means list all the items in the directory.

Functions

  • -a All the items

Autofill

You can use tab after pressing the first few characters. Source
